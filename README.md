# gdg-website

The official website of Google Developer Group on Campus, UNN (GDGoC UNN).

This is a [Next.js](https://nextjs.org) project.

## Getting Started

First, study the UI/UX design on [Figma](https://www.figma.com/design/zLfnmiZAlRb2bfolkbFQNu/GDG?node-id=2-4&t=0oIjPXBIgoMJhcmB-1).

You can get a colour's name from [color-name.com](https://www.color-name.com/hex/4285f4).

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:4096](http://localhost:4096) with your browser to see the result.
